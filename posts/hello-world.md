This is the first post sourced from the `neilk17/public-writing` GitHub repository.

I wanted to share more of my writing in public but there was too much friction in sharing it on other platforms. I tried using Substack and Bearblog but I always wanted to create a place where I owned the stack.

So now I've setup this public repository on GitHub that will publish my writing as soon as it gets committed and pushed.

The next step is to share the same writing on other platforms like Bearblog, Substack and X.
